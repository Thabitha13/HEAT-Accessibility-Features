# HEAT Accessibility Features

Group software engineering project (University of Kent, COMP7024).

Implemented accessibility features for HEAT (Haskell Educational Advanced Tool) IDE:
- Colour-blind mode using Wong palette (deuteranopia support)
- Font size accessibility panel
- Icon enlargement

**Technologies:** Java, Swing, JUnit

**Note:** Original institutional repository at git.cs.kent.ac.uk/comp7024/g5 (requires university credentials)
